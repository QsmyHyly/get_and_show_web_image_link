# 网页图片链接获取和展示扩展

这是一个谷歌插件，功能是在右键菜单和点击插件各自添加一个按钮，功能是获取整个页面所有的图片链接，并打开一个新页面排列出来。这个项目使用了AI。

## 功能特性

- **一键获取**：通过右键菜单或点击扩展图标，一键获取当前网页的所有图片链接
- **智能展示**：在结果页面以网格或列表形式展示所有图片
- **图片预览**：支持图片预览，无需打开新标签页即可查看
- **智能搜索**：支持按URL关键词搜索图片
- **分类筛选**：支持按文件类型或域名分类展示图片
- **多种排序**：支持按文件名、文件类型、域名或图片尺寸排序
- **视图切换**：支持网格视图和列表视图切换
- **批量选择**：支持全选、取消全选和手动选择图片
- **批量下载**：支持下载选中的图片
- **ZIP打包下载**：支持将选中的图片打包成ZIP文件下载
- **新标签页打开**：支持在新标签页中打开选中的图片
- **URL复制**：支持复制选中图片的URL或所有图片的URL
- **详细信息**：显示图片的尺寸、格式和大小信息

## 安装方法

1. 克隆或下载此项目到本地
2. 打开Chrome浏览器，输入 `chrome://extensions` 进入扩展管理页面
3. 启用"开发者模式"
4. 点击"加载已解压的扩展程序"，选择项目目录
5. 扩展安装完成

## 使用说明

1. 访问任意网页
2. 点击扩展图标或在网页上右键选择"获取页面所有图片"
3. 系统会自动扫描页面，获取所有图片链接
4. 扫描完成后会在新标签页中打开结果页面，展示所有图片
5. 在结果页面可以进行搜索、筛选、排序、选择和下载等操作

## 高级功能

### 图片收集管理器
- 自动停止功能：如果一段时间内没有新图片被发现，会自动停止收集
- 手动停止功能：用户可以随时停止图片收集过程
- 持续收集：支持持续监听页面上的新图片

### ZIP打包下载
- 选中多张图片后，可使用"打包下载ZIP"功能
- 系统会将选中的图片打包成ZIP文件供下载
- 支持显示保存对话框，让用户选择保存位置

### 新标签页打开
- 选中多张图片后，可使用"新标签页打开"功能
- 支持在新标签页中批量打开选中的图片
